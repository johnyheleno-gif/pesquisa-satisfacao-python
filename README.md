# Pesquisa de Satisfação

## Descrição
Programa em Python que realiza uma pesquisa com 50 entrevistados para avaliar o atendimento.

## Funcionamento
O sistema coleta:
- Nome
- Idade
- Opinião (1 = Excelente, 2 = Bom, 3 = Ruim)

Ao final, exibe:
- Quantidade de respostas EXCELENTE
- Quantidade de respostas RUIM

## Como executar
Execute no terminal:

python pesquisa.py

## Teste realizado
Foram realizados testes com 10 entrevistados para validar o funcionamento do programa.
